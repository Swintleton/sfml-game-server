# SFML C++ Game Server

Server-side component for the **SFML C++ Game** project. This repository contains the backend/server part of the project and is intended to work together with the game/client repository.

## Showcase Video

A video of the overall project can be found here:  
[SFML C++ Game](https://youtu.be/_CI44NlJ7OY)

## About the Project

This repository contains the server part of a larger C++ game project. While the client repository focuses on gameplay, rendering, and user interaction, this repository is dedicated to the server-side logic and project structure needed to support the game.

The project was built as a more substantial C++ exercise and personal showcase, with a focus on:

- structuring a multi-repository project
- separating client and server responsibilities
- working with C++ beyond small academic examples
- building and maintaining a more complex codebase

## Getting Started

### Prerequisites

- A C++ compiler
- SFML installed and configured
- Visual Studio (recommended, based on the repository contents)

### Build / Run

1. Clone the repository
2. Open the solution/project in Visual Studio
3. Make sure SFML is properly configured
4. Build and run the server project
